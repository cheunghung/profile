My Hobbies


Programming

After the COVID pandemic started in 2020 in HK, I have spent most of my time working from home. Since I've spent more time at home, I was able to fork out extra time to learn how to code. Therefore I have been coding for 4+ years now.

I first started to learn VBA & Python programming since their syntax is easier to learn for a novice programmer. Then I learned RPA proramming and was able to automate some mundane & repetitive tasks at work. If you have worked in KYC onboarding before, you'll realize that most of the time is spent on tedious tasks like name screening, negative news search, and data input. So knowing RPA is a real game changer.

Afterwards, my interest in programming quickly grew. I decided to learn Java due to its popularity in Object Oriented Programming. Also, I am currently building my skills in web development.

Financial Analysis

As a KYC Specialist, being able to analyze financial data is an important skill. For example, when establishing a client's profile, calculating total assets from source of funds & wealth and analyzing a company's financial statements are often required in KYC onboarding.

On the other hand, I am numerically senstive & great in dealing with numbers so I decided to become a financial analyst.
